# 게임 상품 관리 테스트 케이스

## TC-GPROD-001: 게임 상품 목록 조회

- **우선순위**: High
- **유형**: 기능
- **사전조건**: 상품이 연결된 게임이 존재함
- **테스트 절차**:
  1. `/games` 페이지에서 특정 게임의 액션 메뉴 클릭
  2. "상품 관리" 또는 게임 제목 클릭
  3. `/games/{id}/products` 페이지 접속
- **기대결과**: 게임에 연결된 상품 목록이 카드 형태로 표시됨. 각 카드에 순서 번호, 그랜드프라이즈 여부, 상품명, 브랜드, 수량, 지역 배지가 표시됨. 상단에 총 상품수/그랜드프라이즈/지역수 통계 카드 표시됨
- **관련 API**: `GET /admin/game-product-management/games/{gameId}/products/all-details`

---

## TC-GPROD-002: 기존 상품을 게임에 연결 (Link)

- **우선순위**: High
- **유형**: 기능
- **사전조건**: 상품이 존재하고 게임 상품 페이지가 열려 있음
- **테스트 절차**:
  1. "상품 추가" 버튼 클릭
  2. LinkProductToGameDialog 다이얼로그 열림
  3. 기존 상품 목록에서 연결할 상품 선택
  4. 그랜드프라이즈 여부, 수량, 순서 설정
  5. 저장 버튼 클릭
- **기대결과**: 다이얼로그가 닫히고 성공 Toast 표시. 게임 상품 목록이 갱신되어 연결된 상품이 나타남
- **관련 API**: `POST /admin/game-product-management/games/{gameId}/products/{productId}/link`

---

## TC-GPROD-003: 통합 생성 (새 상품 + 슬롯 + 지역 한번에)

- **우선순위**: High
- **유형**: 기능
- **사전조건**: 게임 상품 페이지가 열려 있음
- **테스트 절차**:
  1. "상품 추가" 버튼 클릭
  2. 새 상품 생성 탭 선택 (구현된 경우)
  3. 상품 정보(이름, SKU, 브랜드), 슬롯 정보(수량, 순서), 지역 정보(국가코드, 가격) 모두 입력
  4. 저장 버튼 클릭
- **기대결과**: 상품 생성, 게임 슬롯 등록, 지역 설정이 한 번에 완료됨. 성공 Toast 표시
- **관련 API**: `POST /admin/game-product-management/games/{gameId}/products/complete`

## TC-GPROD-004: 게임에서 상품 제거

- **우선순위**: High
- **유형**: 기능
- **사전조건**: 게임에 연결된 상품이 존재함
- **테스트 절차**:
  1. 게임 상품 페이지에서 특정 상품 카드의 액션 메뉴 클릭
  2. "상품 제거" 선택
  3. 확인 다이얼로그에서 "제거" 버튼 클릭
- **기대결과**: 확인 다이얼로그에 상품명이 표시됨. 삭제 성공 Toast 표시. 상품 목록에서 해당 상품이 제거됨
- **관련 API**: `DELETE /admin/game-product-management/game-products/{gameProductId}/complete`

## TC-GPROD-005: 그랜드프라이즈 설정 토글

- **우선순위**: High
- **유형**: 기능
- **사전조건**: 게임에 상품이 연결되어 있음
- **테스트 절차**:
  1. 그랜드프라이즈가 아닌 상품의 액션 메뉴 클릭
  2. "그랜드 프라이즈 설정" 선택
  3. 토스트 확인 후 상품 카드 상태 확인
- **기대결과**: 해당 상품 카드에 "그랜드 프라이즈" Badge가 추가됨. 성공 Toast 표시
- **관련 API**: `PUT /admin/game-product-management/games/{gameId}/grand-prizes`

## TC-GPROD-007: 연결된 상품 없을 때 빈 상태 표시

- **우선순위**: Medium
- **유형**: UI
- **사전조건**: 상품이 연결되지 않은 게임이 존재함
- **테스트 절차**:
  1. 상품이 없는 게임의 `/games/{id}/products` 페이지 접속
- **기대결과**: EmptyState 컴포넌트가 표시됨: "연결된 상품이 없습니다" 메시지와 "상품 추가" 버튼
- **관련 API**: `GET /admin/game-product-management/games/{gameId}/products/all-details` → 빈 배열

## TC-GPROD-009: 순서 재정렬 (드래그앤드롭)

- **우선순위**: Medium
- **유형**: 기능
- **사전조건**: 게임에 2개 이상의 상품이 연결되어 있음
- **테스트 절차**:
  1. 게임 상품 페이지에서 상품 카드 드래그
  2. 다른 위치로 드롭
- **기대결과**: 상품 순서가 변경됨. sequence 번호가 업데이트됨

> ⚠️ 미구현: 현재 드래그앤드롭 UI 기능이 구현되어 있지 않음. sequence 값은 카드의 `#순서` 번호로 표시만 됨
- **관련 API**: `PUT /admin/game-product-management/games/{gameId}/products/reorder`

## TC-GPROD-010: 게임 상품 슬롯 조회

- **우선순위**: Medium
- **유형**: 기능
- **사전조건**: 게임 상품 슬롯이 존재함
- **테스트 절차**:
  1. 게임 상품 페이지에서 상품 카드의 지역 정보 확인
- **기대결과**: 지역이 설정된 경우 상품 카드 하단에 국가코드/지역코드 Badge가 표시됨 (최대 3개, 이후 +N으로 표시)
- **관련 API**: `GET /admin/game-product-management/game-products/{id}`
