# BlockPick Admin 기능 기획 문서

## 📊 현재 상태 분석

### ✅ 구현 완료된 기능

#### 1. **사이드바 네비게이션**
- Dashboard
- Users
- Games
- Settings

#### 2. **Dashboard 페이지**
- ✅ 기본 통계 카드 (사용자, 게임, 상품, 시스템 상태)
- ✅ 시스템 헬스 체크
- ✅ 시스템 메트릭 표시
- ✅ 최근 게임 목록
- ✅ 최근 사용자 목록
- ❌ 차트 데이터 (API는 있으나 UI 없음)
- ❌ 활동 내역 (API는 있으나 UI 없음)

#### 3. **Users 페이지**
- ✅ 사용자 목록 조회 (테이블)
- ✅ 검색 기능
- ✅ 페이지네이션
- ✅ 사용자 생성 (다이얼로그 완료)
- ✅ 사용자 상세보기 (모달 완료)
- ✅ 사용자 수정 (다이얼로그 완료)
- ✅ 사용자 삭제 (확인 다이얼로그 포함 완료)
- ✅ 사용자 역할 변경 (다이얼로그 완료)
- ❌ 사용자 통계 (API 있음, UI 없음)
- ❌ 사용자 필터링 (API 있음, UI 없음)

#### 4. **Games 페이지**
- ✅ 게임 목록 조회 (테이블)
- ✅ 게임 생성 다이얼로그 (멀티 스텝: Step 1 게임 정보 생성 → Step 2 상품 연결)
- ✅ 검색 기능
- ✅ 페이지네이션
- ❌ 게임 상세보기 (API 있음, UI 없음)
- ❌ 게임 수정 (API 있음, UI 없음)
- ❌ 게임 삭제 (API 있음, 기능 없음)
- ❌ 게임 강제 종료 (API 있음, 기능 없음)
- ❌ 게임 리더보드 (API 있음, UI 없음)
- ❌ 게임 통계 (API 있음, UI 없음)
- ❌ 게임 필터링 (타입, 상태별) (API 있음, UI 없음)

#### 5. **Settings 페이지**
- ✅ 기본 UI 구조
- ❌ 프로필 수정 기능 (API 없음)
- ❌ 비밀번호 변경 기능 (API 없음)
- ❌ 알림 설정 기능 (API 없음)
- ❌ 시스템 설정 기능 (API 없음)

## 🚫 완전히 누락된 주요 기능 영역

### 1. **Products 관리** (완전히 없음)
- API: ✅ 모두 구현됨
- UI: ❌ 없음
- 필요한 페이지:
  - `/products` - 상품 목록 페이지
  - `/products/[id]` - 상품 상세 페이지
  - 상품 생성/수정/삭제 기능
  - 상품 카테고리 관리
  - 상품 통계

### 2. **Game-Product 관리** (완전히 없음)
- API: ✅ 모두 구현됨
- UI: ❌ 없음
- 필요한 페이지:
  - 게임별 상품 관리 페이지
  - 상품-게임 연결 관리
  - 지역별 상품 설정
  - 그랜드 프라이즈 설정
  - 상품 순서 관리

### 3. **Blockchain 관리** (완전히 없음)
- API: ✅ 모두 구현됨
- UI: ❌ 없음
- 필요한 페이지:
  - 컨트랙트 배포 상태 모니터링
  - 트랜잭션 조회
  - 엔트리 상태 조회
  - 블록체인 이벤트 모니터링

### 4. **Storage/File 관리** (완전히 없음)
- API: ✅ 모두 구현됨
- UI: ❌ 없음
- 필요한 페이지:
  - 파일 업로드 UI
  - 파일 목록 조회
  - 파일 관리 (삭제, 정보 조회)
  - 폴더별 파일 관리

### 5. **Monitoring** (완전히 없음)
- API: ✅ 모두 구현됨
- UI: ❌ 없음
- 필요한 페이지:
  - 시스템 로그 조회
  - 성능 메트릭 대시보드
  - 이벤트 상태 모니터링
  - 트랜잭션 상태 모니터링
  - 실패한 이벤트 재시도

## 📝 페이지별 상세 기능 명세

### Users 페이지 상세 기능

#### 사용자 생성 다이얼로그
**필드**:
- 이메일 (필수, 이메일 형식 검증)
- 비밀번호 (필수, 최소 8자)
- 닉네임 (선택)
- 역할 (필수, 드롭다운: USER, ADMIN 등)
- 프로필 이미지 (선택, 파일 업로드)

**동작**:
1. 폼 제출 시 `userService.createUser()` 호출
2. 성공 시 토스트 알림 및 목록 새로고침
3. 실패 시 에러 메시지 표시

#### 사용자 상세보기 모달
**표시 정보**:
- 기본 정보 (이메일, 닉네임, 역할, 가입일)
- 포인트/캐시 잔액
- 소셜 계정 정보 (연결된 경우)
- 계정 상태
- 최근 활동 (선택적)

**액션 버튼**:
- 수정
- 역할 변경
- 정지/해제
- 삭제

#### 사용자 수정 다이얼로그
**수정 가능 필드**:
- 이메일
- 닉네임
- 프로필 이미지

**동작**:
- `userService.updateUser()` 호출

#### 사용자 삭제 기능
**플로우**:
1. Actions 메뉴에서 "Delete" 클릭
2. 확인 다이얼로그 표시
3. 확인 시 `userService.deleteUser()` 호출
4. 성공 시 목록에서 제거

#### 역할 변경 기능
**플로우**:
1. Actions 메뉴에서 "Change Role" 클릭
2. 역할 선택 드롭다운 표시
3. 선택 시 `userService.updateUserRole()` 호출
4. 즉시 반영

#### 필터링 UI
**필터 옵션**:
- 상태: 전체 / ACTIVE / INACTIVE / SUSPENDED / DELETED
- 역할: 전체 / USER / ADMIN / 기타
- 가입일: 날짜 범위 선택
- 검색: 이메일/닉네임 검색

**동작**:
- 필터 변경 시 즉시 API 호출 및 목록 업데이트
