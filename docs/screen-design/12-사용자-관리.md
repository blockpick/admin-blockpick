## 섹션별 상세

### PageHeader

| 항목 | 내용 |
|------|------|
| title | 사용자 관리 |
| description | 시스템 사용자를 관리합니다 |
| action | label: "사용자 추가", icon: UserPlus, onClick: CreateUserDialog 열기 |

---

### 통계 카드 그리드

`grid gap-4 md:grid-cols-2 lg:grid-cols-4`

| 순서 | title | icon | description |
|------|-------|------|-------------|
| 1 | 총 사용자 | `Users` | - |
| 2 | 활성 사용자 | `UserCheck` | - |
| 3 | 정지 사용자 | `UserX` | - |
| 4 | 비활성 사용자 | `UserMinus` | - |

---

### UserFilterBar

**필터 항목**:

| 필드 | 컴포넌트 | 옵션 |
|------|----------|------|
| 상태 | Select | 전체 / ACTIVE / SUSPENDED / INACTIVE |
| 역할 | Select | 전체 / UserRole enum 값들 |
| 검색어 | Input | 이름, 이메일 검색 |
| 시작일 | DateTimePicker | - |
| 종료일 | DateTimePicker | - |

- 확장/축소 토글 버튼
- "필터 초기화" 버튼
- 활성 필터 수 Badge 표시

---

### DataTable

**컬럼 정의**:

| 컬럼 | 내용 | 정렬 |
|------|------|------|
| 사용자 | Avatar + 닉네임 + 이메일 | - |
| 역할 | Badge (role) | 가능 |
| 상태 | Badge (status) | 가능 |
| 가입일 | 날짜 포맷 | 가능 |
| 액션 | DropdownMenu | - |

**페이지네이션**: `enableServerSidePagination=true`, 서버 API 호출

**검색**: `searchKey="email"` (서버사이드이므로 UserFilterBar의 search 필드 사용)

**액션 DropdownMenu 항목**:
- 상세 보기 → UserDetailDialog 열기
- 수정 → EditUserDialog 열기
- 역할 변경 → RoleChangeDialog 열기
- 정지/활성화 → ConfirmDialog (상태에 따라 텍스트 분기)
- 삭제 → ConfirmDialog (destructive)

---

## 다이얼로그 목록

| 다이얼로그 | 트리거 | 컴포넌트 |
|-----------|--------|----------|
| 사용자 생성 | PageHeader action 버튼 | `CreateUserDialog` |
| 사용자 상세 | 액션 메뉴 "상세 보기" | `UserDetailDialog` |
| 사용자 수정 | 액션 메뉴 "수정" / UserDetailDialog "수정" 버튼 | `EditUserDialog` |
| 역할 변경 | 액션 메뉴 "역할 변경" | `ConfirmDialog` |
| 상태 변경 | 액션 메뉴 "정지/활성화" | `ConfirmDialog` |
| 삭제 확인 | 액션 메뉴 "삭제" | `ConfirmDialog (destructive)` |

## 데이터 연동

| Hook | 역할 |
|------|------|
| `useUserStats()` | 통계 카드 데이터 |
| `useUsers(filters, pagination)` | 사용자 목록 (페이지네이션, 필터) |
| `useDeleteUser()` | 사용자 삭제 mutation |
| `useUpdateUserStatus()` | 상태 변경 mutation |
| `useUpdateUserRole()` | 역할 변경 mutation |
