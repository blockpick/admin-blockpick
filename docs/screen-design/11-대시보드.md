## 화면 구조

```
AdminLayout
└── div.space-y-6
    ├── PageHeader
```

## 섹션별 상세

### PageHeader

| 항목 | 내용 |
|------|------|

| description | BlockPick 관리자 패널 개요 |

---

### 통계 카드 그리드

`grid gap-4 md:grid-cols-2 lg:grid-cols-4`

| 순서 | title | value 소스 | icon | description |
|------|-------|-----------|------|-------------|
| 1 | 총 사용자 | `stats.totalUsers` | `Users` | `활성: {stats.activeUsers}` |
| 2 | 전체 게임 | `stats.totalGames` | `Gamepad2` | `진행중: {stats.activeGames}` |
| 3 | 전체 상품 | `stats.totalProducts` | `Package` | `활성: {stats.activeProducts}` |
| 4 | 시스템 상태 | `stats.systemStatus` | `Activity` | `정상 운영중` |

- 로딩 중: StatsCard 자체 Skeleton 처리

---

### 시스템 알림 + 시스템 메트릭

`grid gap-4 md:grid-cols-2`

#### 시스템 알림 (Card)

- `CardHeader`: 제목 "시스템 알림", Bell 아이콘
- `CardContent`: 알림 목록 (ActivityTimeline 형태)
  - 각 알림: 아이콘 원 + 메시지 + 시간
  - 알림 없음: EmptyState

#### 시스템 메트릭 (Card)

- `CardHeader`: 제목 "시스템 메트릭"
- `CardContent`: 메트릭 항목 목록
  - CPU 사용률, 메모리 사용률, 응답시간, 업타임
  - 각 항목: 레이블 + 수치 + 진행 바(Progress)

---

### 차트 그리드

`grid gap-4 md:grid-cols-2`

| 순서 | title | 데이터 소스 | 기간 선택 |
|------|-------|-----------|----------|
| 1 | 사용자 추이 | `chartData.users` | 있음 (일/주/월/년) |
| 2 | 게임 활동 | `chartData.games` | 있음 |
| 3 | 수익 현황 | `chartData.revenue` | 있음 |
| 4 | 상품 현황 | `chartData.products` | 있음 |

- ChartCard 컴포넌트, Recharts LineChart

---

### ActivityTimeline + 최근 게임 목록

`grid gap-4 md:grid-cols-2`

#### 최근 활동 (ActivityTimeline)

- `activities`: 최근 시스템 활동 데이터
- `limit`: 10개
- 타임라인 UI (아이콘 원 + 세로선 + 텍스트)

#### 최근 게임 목록 (Card)

- `CardHeader`: 제목 "최근 게임", "전체 보기" 링크(→ /games)
- `CardContent`: 게임 목록
  - 각 항목: 게임명 + 상태 Badge + 날짜
  - 최대 5개 표시

---

### 최근 사용자 목록 (전체 너비)

- Card 형태
- `CardHeader`: 제목 "최근 가입 사용자", "전체 보기" 링크(→ /users)
- `CardContent`: DataTable (컬럼: Avatar+이름, 이메일, 역할, 가입일, 상태)
- 페이지네이션 없음, 최근 5~10개만 표시

---

## 데이터 연동

| Hook | 반환 데이터 |
|------|------------|
| `useDashboardStats()` | 4개 통계 수치 |
| `useSystemAlerts()` | 시스템 알림 목록 |
| `useSystemMetrics()` | CPU/메모리/응답시간 등 |
| `useDashboardCharts()` | 차트 시계열 데이터 |
| `useRecentActivities()` | 최근 활동 목록 |
| `useRecentGames()` | 최근 게임 목록 |
| `useRecentUsers()` | 최근 사용자 목록 |
