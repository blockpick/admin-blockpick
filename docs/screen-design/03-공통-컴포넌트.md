## 1. UI 기본 컴포넌트 (shadcn/ui 기반)

### Button

**경로**: `src/components/ui/button.tsx`

| Prop | 타입 | 기본값 | 설명 |
|------|------|--------|------|
| `variant` | `'default' \| 'destructive' \| 'outline' \| 'secondary' \| 'ghost' \| 'link'` | `'default'` | 시각적 스타일 |
| `size` | `'default' \| 'sm' \| 'lg' \| 'icon'` | `'default'` | 크기 |
| `asChild` | `boolean` | `false` | Radix Slot으로 자식 컴포넌트에 렌더링 |

**Variant 스펙**:

| Variant | 배경 | 텍스트 | 호버 | 사용처 |
|---------|------|--------|------|--------|
| `default` | `bg-primary` | `text-primary-foreground` | `bg-primary/90` | 주요 액션 |
| `destructive` | `bg-destructive` | `text-destructive-foreground` | `bg-destructive/90` | 삭제/위험 |
| `outline` | 투명 | `text-foreground` | `bg-accent` | 보조 액션 |
| `secondary` | `bg-secondary` | `text-secondary-foreground` | `bg-secondary/80` | 중립 액션 |
| `ghost` | 투명 | `text-foreground` | `bg-accent` | 아이콘 버튼, 네비게이션 |
| `link` | 투명 | `text-primary` | underline | 텍스트 링크 |

**Size 스펙**:

| Size | 높이 | 패딩 | 용도 |
|------|------|------|------|
| `default` | `h-10` | `px-4 py-2` | 일반 버튼 |
| `sm` | `h-9` | `px-3` | 작은 버튼 (테이블 액션 등) |
| `lg` | `h-11` | `px-8` | 큰 버튼 (폼 제출 등) |
| `icon` | `h-10 w-10` | - | 아이콘 전용 (정사각형) |

- 검색바 (searchKey 있을 때만 표시)
- 페이지네이션 UI: 이전/다음 버튼 + 페이지 번호 (7개 이하: 전체 표시, 초과: 생략 부호)

```
        ├── 기본 정보 섹션

        └── 메타데이터 섹션
```

### Edit Dialogs (공통 패턴)

`EditUserDialog`, `EditGameDialog`, `EditProductDialog`는 공통 구조를 따릅니다:

- React Hook Form + Zod 유효성 검사
- 기존 데이터로 폼 초기화 (`useEffect` + `form.reset()`)
- 저장 성공 시 Toast + 다이얼로그 닫기
- 저장 실패 시 Toast(destructive)
