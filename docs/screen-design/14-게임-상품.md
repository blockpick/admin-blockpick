# 14. 게임 상품 관리

## 기본 정보

| 항목 | 내용 |
|------|------|
| URL | `/games/[id]/products` |
| 파일 | `src/app/games/[id]/products/page.tsx` |
| 레이아웃 | AdminLayout |
| 인증 | 필요 |
| 라우트 파라미터 | `id`: 게임 ID |

`grid gap-4 md:grid-cols-3` (statsLoading=false && stats 존재 시만 표시)

| 순서 | title | value | icon | description |
|------|-------|-------|------|-------------|
| 1 | 총 상품 수 | `stats.total` | `Package` | `활성: {stats.active}` |

### 상품 카드 그리드

`grid gap-4 md:grid-cols-2 lg:grid-cols-3`

**상품 없음**: `EmptyState` 표시

- description: "게임에 상품을 추가하여 시작하세요"

**상품 있음**: 각 상품 Card

### 상품 Card 구조

```
Card (relative)
└── CardContent.p-4
    └── div.flex.items-start.gap-3
        ├── 이미지 영역 (h-16 w-16, flex-shrink-0)
        │   ├── 이미지 있음: Image (next/image, fill, object-cover, rounded-lg)
        │   └── 이미지 없음: div (border-dashed, Package 아이콘)
        └── div.flex-1
            ├── 상단: 시퀀스 + Badge들 + DropdownMenu
            │   ├── "#{product.sequence || index+1}" (text-xs text-muted-foreground)
            │   ├── 그랜드프라이즈 Badge (Trophy 아이콘, variant="default") - isGrandPrize 시만
            │   ├── 비활성 Badge (variant="secondary") - !active 시만
            │   └── DropdownMenu (MoreHorizontal 아이콘)
            ├── 상품명 (font-medium text-sm truncate)
            ├── 브랜드명 (text-xs text-muted-foreground) - brand 있을 때만
            ├── 수량 (text-xs text-muted-foreground) - quantity 있을 때만
            └── 지역 Badge들 - regions 있을 때만
                ├── 최대 3개 Badge (countryCode || regionCode)
                └── 3개 초과 시 "+N" Badge
```

## 그랜드 프라이즈 토글 로직

```typescript
// 현재 상품의 isGrandPrize 반전 후,
// 전체 grandPrize 상품 ID 목록 재계산하여 일괄 설정
const grandPrizeIds = products
  ?.filter((p) => (p.id === productId ? !currentStatus : p.isGrandPrize))
  .map((p) => p.id) || [];

await setGrandPrizes.mutateAsync({
  gameId,
  request: { gameProductIds: grandPrizeIds },
});
```

## 로딩/오류 상태

| 상태 | 표시 |
|------|------|
| gameLoading 또는 productsLoading | `LoadingSpinner size="lg"` (h-screen 중앙) |
| game 데이터 없음 | `EmptyState` (Package 아이콘, "게임을 찾을 수 없습니다") |
| 상품 목록 비어있음 | `EmptyState` (상품 추가 CTA 포함) |

## 다이얼로그

| Prop | 값 |
|------|-----|
| `gameId` | URL 파라미터 id |
| `open` | `linkDialogOpen` state |
| `onOpenChange` | `setLinkDialogOpen` |

### ConfirmDialog (상품 제거)

| Prop | 값 |
|------|-----|
| `title` | "상품 제거" |
| `description` | `정말로 "{selectedProductName}" 상품을 게임에서 제거하시겠습니까?` |
| `confirmText` | "제거" |
| `cancelText` | "취소" |
| `variant` | `"destructive"` |
| `onConfirm` | `handleConfirmDelete` |

## 데이터 연동

| Hook | 역할 |
|------|------|
| `useGame(gameId)` | 게임 기본 정보 |
