# 18. 파일 관리

## 기본 정보

| 항목 | 내용 |
|------|------|
| URL | `/storage` |
| 파일 | `src/app/storage/page.tsx` |
| 레이아웃 | AdminLayout |
| 인증 | 필요 |

---

## 화면 구조

```
AdminLayout
└── div.space-y-6
    ├── PageHeader
    ├── 통계 카드 그리드 (4열)
    ├── 폴더 선택 + 업로드 버튼 영역
    ├── 파일 그리드 (4열) 또는 EmptyState
    ├── 파일 미리보기 Dialog
    ├── 업로드 Dialog
    └── 삭제 ConfirmDialog
```

## 섹션별 상세

### PageHeader

| 항목 | 내용 |
|------|------|
| title | 파일 관리 |
| description | 업로드된 파일을 관리합니다 |
| action | label: "파일 업로드", icon: Upload, onClick: 업로드 Dialog 열기 |

---

### 통계 카드 그리드

`grid gap-4 md:grid-cols-2 lg:grid-cols-4`

| 순서 | title | icon | description |
|------|-------|------|-------------|
| 1 | 총 파일 수 | `Files` | - |
| 2 | 이미지 파일 | `Image` | - |
| 3 | 기타 파일 | `File` | - |
| 4 | 총 용량 | `HardDrive` | 사용 중인 스토리지 |

---

### 폴더 선택 + 업로드 버튼 영역

```
div.flex.items-center.justify-between
├── Select (폴더 선택)
│   └── 옵션: 전체 / products / games / profiles / ...
└── Button "파일 업로드" (Upload 아이콘)
```

---

### 파일 그리드

`grid gap-4 grid-cols-2 md:grid-cols-3 lg:grid-cols-4`

**파일 없음**: EmptyState
- icon: `FolderOpen`
- title: "파일이 없습니다"
- description: "파일을 업로드하여 시작하세요"

**파일 있음**: 각 파일 Card

---

### 파일 Card 구조

```
Card (cursor-pointer, hover:shadow-md)
└── CardContent.p-3
    ├── 미리보기 영역 (aspect-square, rounded-md, overflow-hidden)
    │   ├── 이미지 파일: next/image (object-cover)
    │   └── 기타 파일: File 아이콘 (중앙)
    ├── 파일명 (text-sm font-medium truncate, mt-2)
    ├── 파일 크기 (text-xs text-muted-foreground)
    └── 액션 버튼 영역
        ├── 미리보기 버튼 (Eye 아이콘)
        └── 삭제 버튼 (Trash2 아이콘, text-destructive)
```

- Card 클릭 → 파일 미리보기 Dialog 열기
- 파일 크기: bytes → KB/MB 변환 표시

---

## 파일 미리보기 Dialog

```
Dialog (max-w-2xl)
└── DialogContent
    ├── DialogHeader
    │   ├── DialogTitle: 파일명
    │   └── DialogDescription: 파일 크기 + 업로드 날짜
    └── 본문
        ├── 이미지 파일: next/image (전체 너비)
        └── 기타 파일: 파일 정보 표시
            └── (파일 타입 아이콘 + 메타데이터)
        └── DialogFooter
            ├── Button "다운로드" (Download 아이콘)
            └── Button "삭제" (destructive)
```

---

## 업로드 Dialog

```
Dialog
└── DialogContent
    ├── DialogHeader: "파일 업로드"
    └── 본문
        ├── 폴더 선택 (Select)
        ├── 드래그앤드롭 영역
        │   └── 또는 클릭하여 파일 선택
        ├── 선택된 파일 목록 (파일명 + 크기 + 제거 버튼)
        └── DialogFooter
            ├── Button "취소"
            └── Button "업로드" (진행 중: disabled + 스피너)
```

## 데이터 연동

| Hook | 역할 |
|------|------|
| `useStorageStats()` | 통계 카드 데이터 |
| `useStorageFiles(folder)` | 파일 목록 (폴더 필터) |
| `useStorageFolders()` | 폴더 목록 |
| `useUploadFile()` | 파일 업로드 mutation |
| `useDeleteFile()` | 파일 삭제 mutation |
