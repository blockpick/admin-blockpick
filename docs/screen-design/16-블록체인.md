### DataTable (컨트랙트 목록)

**컬럼 정의**:

| 컬럼 | 내용 | 정렬 |
|------|------|------|
| 게임 | 게임명 | - |
| 컨트랙트 주소 | contractAddress (모노스페이스, 축약) | - |
| 네트워크 | network | 가능 |
| 배포 상태 | Badge (deployStatus) | 가능 |
| 배포일 | deployedAt | 가능 |
| 액션 | DropdownMenu | - |

**액션 DropdownMenu 항목**:
- 트랜잭션 상세 → 트랜잭션 상세 Dialog 열기
- 컨트랙트 주소 복사 → clipboard API

---

## 배포 상태 Badge

| 상태 | Variant | 표시 텍스트 |
|------|---------|------------|
| DEPLOYED | `default` | 배포 완료 |
| PENDING | `secondary` | 배포 대기 |
| FAILED | `destructive` | 배포 실패 |
| DEPLOYING | `outline` | 배포 중 |

---

## 트랜잭션 상세 Dialog

**표시 정보**:

| 항목 | 내용 |
|------|------|
| 트랜잭션 해시 | txHash (모노스페이스, 복사 버튼) |
| 블록 번호 | blockNumber |
| 상태 | Badge (status) |
| 가스 사용량 | gasUsed |
| 타임스탬프 | timestamp |
| From | from 주소 |
| To | to 주소 (컨트랙트) |
| 이벤트 로그 | 이벤트 목록 |

---

## 엔트리 상태 Dialog

**표시 정보**:

| 항목 | 내용 |
|------|------|
| 엔트리 ID | entryId |
| 게임 | 게임명 |
| 사용자 | 사용자명 |
| 상태 | Badge (entryStatus) |
| 블록체인 TX | txHash (있을 경우) |
| 참가 시간 | createdAt |

## 데이터 연동

| Hook | 역할 |
|------|------|
| `useBlockchainStats()` | 통계 카드 데이터 |
| `useContracts(pagination)` | 컨트랙트 목록 |
| `useTransactionByHash(hash)` | 트랜잭션 해시 조회 |

## 특이사항

- 컨트랙트 주소 및 트랜잭션 해시는 `font-mono text-xs` 로 표시
- 긴 해시값은 앞 6자 + `...` + 뒤 4자 형태로 축약 표시
- 외부 블록체인 탐색기 링크 (예: Etherscan) 연결 옵션 제공 가능
