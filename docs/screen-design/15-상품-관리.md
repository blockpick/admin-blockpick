# 15. 상품 관리

## 기본 정보

| 항목 | 내용 |
|------|------|
| URL | `/products` |
| 파일 | `src/app/products/page.tsx` |
| 레이아웃 | AdminLayout |
| 인증 | 필요 |

---

## 화면 구조

```
AdminLayout
└── div.space-y-6
    ├── PageHeader
    ├── 통계 카드 그리드 (4열)
    ├── ProductFilterBar
    └── DataTable (서버사이드 페이지네이션)
```

---

## 섹션별 상세

### PageHeader

| 항목 | 내용 |
|------|------|
| title | 상품 관리 |
| description | 상품을 등록하고 관리합니다 |
| action | label: "상품 추가", icon: Plus, onClick: CreateProductDialog 열기 |

---

### 통계 카드 그리드

`grid gap-4 md:grid-cols-2 lg:grid-cols-4`

| 순서 | title | icon | description |
|------|-------|------|-------------|
| 1 | 총 상품 수 | `Package` | - |
| 2 | 활성 상품 | `PackageCheck` | - |
| 3 | 브랜드 수 | `Tag` | - |
| 4 | 카테고리 수 | `Layers` | - |

---

### ProductFilterBar

| Prop | 타입 | 설명 |
|------|------|------|
| `brands` | `string[]` | 브랜드 목록 (Select 옵션) |
| `categories` | `string[]` | 카테고리 목록 (Select 옵션) |

**필터 항목**:

| 필드 | 컴포넌트 | 옵션 |
|------|----------|------|
| 브랜드 | Select | 전체 / brands 배열 |
| 카테고리 | Select | 전체 / categories 배열 |
| 활성 여부 | Select | 전체 / 활성 / 비활성 |

---

### DataTable

**컬럼 정의**:

| 컬럼 | 내용 | 정렬 |
|------|------|------|
| 상품 | 썸네일 + 상품명 | - |
| 브랜드 | brand | 가능 |
| SKU | sku | - |
| 상태 | Badge (active) | 가능 |
| 등록일 | createdAt | 가능 |
| 액션 | DropdownMenu | - |

**페이지네이션**: `enableServerSidePagination=true`

**액션 DropdownMenu 항목**:
- 상세 보기 → ProductDetailDialog 열기
- 수정 → EditProductDialog 열기
- 삭제 → ConfirmDialog (destructive)

---

## 다이얼로그 목록

| 다이얼로그 | 트리거 | 컴포넌트 |
|-----------|--------|----------|
| 상품 생성 | PageHeader action 버튼 | `CreateProductDialog` |
| 상품 상세 | 액션 메뉴 "상세 보기" | `ProductDetailDialog` |
| 상품 수정 | 액션 메뉴 "수정" / ProductDetailDialog "수정" 버튼 | `EditProductDialog` |
| 삭제 확인 | 액션 메뉴 "삭제" | `ConfirmDialog (destructive)` |

## CreateProductDialog 폼 필드

| 필드 | 컴포넌트 | 필수 | 비고 |
|------|----------|------|------|
| 상품명 | Input | 예 | |
| 설명 | Textarea | 아니오 | |
| 브랜드 | Input | 아니오 | |
| SKU | Input | 아니오 | 재고 관리 코드 |
| 썸네일 이미지 | file Input | 아니오 | storageService 업로드 |

## ProductDetailDialog 표시 정보

| 섹션 | 표시 항목 |
|------|----------|
| 이미지 | thumbnailUrl, defaultImage (next/image) |

## 상태 Badge

| active 값 | Variant | 표시 텍스트 |
|-----------|---------|------------|
| `true` | `default` | 활성 |
| `false` | `secondary` | 비활성 |

## 데이터 연동

| Hook | 역할 |
|------|------|
| `useProductStats()` | 통계 카드 데이터 |
| `useProducts(filters, pagination)` | 상품 목록 (서버사이드) |
| `useProductBrands()` | 브랜드 목록 (필터바용) |
| `useProductCategories()` | 카테고리 목록 (필터바용) |
| `useCreateProduct()` | 상품 생성 mutation |
| `useUpdateProduct()` | 상품 수정 mutation |
| `useDeleteProduct()` | 상품 삭제 mutation |
