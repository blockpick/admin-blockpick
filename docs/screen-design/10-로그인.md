```

└── Card (w-full max-w-md)

    └── CardContent

            ├── 비밀번호 필드 (Label + Input type="password")
```

## 컴포넌트 상세

### 로고 영역

- `div.h-12.w-12.rounded-lg.bg-primary`: 브랜드 색상 정사각형 로고
- 위치: `flex justify-center mb-4`

### 이메일 필드

| 속성 | 값 |
|------|-----|
| Label | 이메일 |
| Input type | `email` |
| placeholder | `admin@blockpick.com` |
| required | true |

### 비밀번호 필드

| 속성 | 값 |
|------|-----|
| Label | 비밀번호 |
| Input type | `password` |
| required | true |

### 로그인 버튼

| 상태 | 텍스트 | 비활성 조건 |
|------|--------|------------|
| 기본 | 로그인 | - |
| 요청 중 | 로그인 중... | `login.isPending = true` |

### 오류 알림

- `Alert variant="destructive"` 사용
- API 오류 메시지 우선, 없으면 기본 메시지: `'이메일 또는 비밀번호가 올바르지 않습니다'`
- 오류 없으면 미표시

## 상태 흐름

### 로딩 상태 (인증 확인 중)

```
authLoading = true
→ 전체 화면: LoadingSpinner size="lg"
   (flex min-h-screen items-center justify-center)
```

### 이미 로그인된 경우

```
localStorage('auth_token') 존재 && isAuthenticated = true
→ useEffect → router.push('/dashboard')
```

### 로그인 성공

```
form.submit
→ login.mutateAsync({ email, password })
→ localStorage.setItem('auth_token', response.accessToken)
→ localStorage.setItem('refresh_token', response.refreshToken)
→ router.push('/dashboard')
→ router.refresh()
```

### 로그인 실패

```
login.mutateAsync throw
→ setError(err.message || err.data.message || err.error || 기본 메시지)
→ Alert(destructive) 표시
→ console.error 로깅
```
