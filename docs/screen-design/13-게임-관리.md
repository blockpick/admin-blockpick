---

### DataTable

**컬럼 정의**:

| 컬럼 | 내용 | 정렬 |
|------|------|------|
| 게임명 | title | 가능 |
| 타입 | Badge (type) | 가능 |
| 상태 | Badge (status) | 가능 |
| 참가비 | entryFee + currency | 가능 |
| 기간 | startTime ~ endTime | 가능 |
| 상품 수 | 연결 상품 개수 | - |
| 액션 | DropdownMenu | - |

**페이지네이션**: `enableServerSidePagination=true`

**액션 DropdownMenu 항목**:
- 상세 보기 → GameDetailDialog 열기
- 수정 → EditGameDialog 열기
- 상품 관리 → `/games/{id}/products` 이동
- 리더보드 → `/games/{id}/leaderboard` 이동
- 강제 종료 → ConfirmDialog
- 정산 → ConfirmDialog
- 삭제 → ConfirmDialog (destructive)

## CreateGameDialog 상세 (2-Step)

### Step 1 - 게임 기본 정보

| 필드 | 컴포넌트 | 필수 | 비고 |
|------|----------|------|------|
| 게임명 | Input | 예 | |
| 게임 타입 | Select (GameType) | 예 | |
| 설명 | Textarea | 아니오 | |
| 카테고리 | Input | 아니오 | |
| 참가비 | Input (number) | 아니오 | |
| 통화 | Select | 아니오 | |
| 최소 참가자 | Input (number) | 아니오 | |
| 최대 참가자 | Input (number) | 아니오 | |
| 1인 최대 참가 | Input (number) | 아니오 | |
| 보상 포인트 | Input (number) | 아니오 | |
| 그리드 행 수 | Input (number) | 아니오 | |
| 그리드 열 수 | Input (number) | 아니오 | |
| 노출 시작 | DateTimePicker | 아니오 | |
| 시작 시간 | DateTimePicker | 아니오 | |
| 종료 시간 | DateTimePicker | 아니오 | |
| 공개 여부 | Switch (isPublic) | - | |
| 중복 참가 허용 | Switch (allowMultipleEntries) | - | |
| 자동 정산 | Switch (autoSettle) | - | |
| 블록체인 사용 | Switch (blockchainEnabled) | - | |
| 참가비 사용 | Switch (useEntryFee) | - | |
| 보상 포인트 사용 | Switch (useRewardPoint) | - | |

### Step 2 - 상품 연결

- 상품 검색 Input
- 상품 목록 (체크박스 선택)
- 블록체인 컨트랙트 배포 옵션 Switch
- 배포 진행 중: 5초 폴링으로 상태 확인

## 상태 Badge 색상

| 상태 | Variant | 표시 텍스트 |
|------|---------|------------|
| SCHEDULED | `secondary` | 예정 |
| ACTIVE | `default` | 진행중 |
| COMPLETED | `outline` | 완료 |
| CANCELLED | `destructive` | 취소 |
| SETTLING | `secondary` | 정산중 |

---

## GameDetailDialog 표시 정보

| 섹션 | 표시 항목 |
|------|----------|
| 기본 정보 | title, type, status, category, description |
| 참가/보상 | entryFee, currency, minEntries, maxEntries, maxEntriesPerUser, rewardPoint |
| 시간 | visibleFrom, startTime, endTime |
| 그리드 | gridRows, gridCols |
| 옵션 | isPublic, allowMultipleEntries, autoSettle, blockchainEnabled, useEntryFee, useRewardPoint |
| 연결 상품 | 상품 목록 (Badge 형태) |
| 블록체인 | contractAddress, txHash |

## 데이터 연동

| Hook | 역할 |
|------|------|
| `useGameStats()` | 통계 카드 데이터 |
| `useGames(filters, pagination)` | 게임 목록 |
| `useCreateGame()` | 게임 생성 mutation |
| `useUpdateGame()` | 게임 수정 mutation |
| `useDeleteGame()` | 게임 삭제 mutation |
| `useForceEndGame()` | 강제 종료 mutation |
| `useSettleGame()` | 정산 mutation |
