## 색상 토큰 (Color Tokens)

shadcn/ui 컨벤션에 따라 HSL 값을 CSS 변수로 정의합니다. Tailwind 클래스에서 `bg-background`, `text-foreground` 등으로 사용됩니다.

### 라이트 테마 (`:root`)

| 토큰 변수 | HSL 값 | 용도 |
|-----------|--------|------|
| `--background` | `0 0% 100%` | 페이지 배경 (흰색) |
| `--foreground` | `222.2 84% 4.9%` | 기본 텍스트 (거의 검정) |
| `--card` | `0 0% 100%` | 카드 배경 |
| `--card-foreground` | `222.2 84% 4.9%` | 카드 내 텍스트 |
| `--popover` | `0 0% 100%` | 팝오버/드롭다운 배경 |
| `--popover-foreground` | `222.2 84% 4.9%` | 팝오버 내 텍스트 |
| `--primary` | `222.2 47.4% 11.2%` | 주요 액션 색상 (남색 계열) |
| `--primary-foreground` | `210 40% 98%` | primary 위 텍스트 (밝은 색) |
| `--secondary` | `210 40% 96.1%` | 보조 색상 (연한 회색) |
| `--secondary-foreground` | `222.2 47.4% 11.2%` | secondary 위 텍스트 |
| `--muted` | `210 40% 96.1%` | 흐린 배경 (비활성, 서브 영역) |
| `--muted-foreground` | `215.4 16.3% 46.9%` | 흐린 텍스트 (플레이스홀더, 설명) |
| `--accent` | `210 40% 96.1%` | 호버 강조 배경 |
| `--accent-foreground` | `222.2 47.4% 11.2%` | accent 위 텍스트 |
| `--destructive` | `0 84.2% 60.2%` | 위험/삭제 액션 색상 (빨간색) |
| `--destructive-foreground` | `210 40% 98%` | destructive 위 텍스트 |
| `--border` | `214.3 31.8% 91.4%` | 테두리 색상 (연한 회색) |
| `--input` | `214.3 31.8% 91.4%` | 입력 필드 테두리 |
| `--ring` | `222.2 84% 4.9%` | 포커스 링 색상 |

### Tailwind 클래스 매핑

```
bg-background       → var(--background)
bg-foreground       → var(--foreground)
bg-card             → var(--card)
bg-primary          → var(--primary)
text-primary-foreground → var(--primary-foreground)
bg-secondary        → var(--secondary)
bg-muted            → var(--muted)
text-muted-foreground → var(--muted-foreground)
bg-accent           → var(--accent)
bg-destructive      → var(--destructive)
border-border       → var(--border)
border-input        → var(--input)
ring-ring           → var(--ring)
```

## 테두리 둥글기 (Border Radius)

| 토큰 | 계산값 | Tailwind 클래스 | 사용 예 |
|------|--------|-----------------|---------|
| `--radius` | `0.5rem` (8px) | 기준값 | - |
| `lg` | `var(--radius)` = 0.5rem | `rounded-lg` | Card, Dialog, Input |
| `md` | `calc(var(--radius) - 2px)` = 0.375rem | `rounded-md` | Button (기본), Badge |
| `sm` | `calc(var(--radius) - 4px)` = 0.25rem | `rounded-sm` | 내부 요소 |
| `full` | - | `rounded-full` | Avatar, Badge (pill) |

## 간격 (Spacing)

Tailwind 기본 간격 스케일(4px 기준) 사용. 주요 사용 패턴:

| Tailwind 클래스 | 값 | 주요 사용처 |
|-----------------|-----|------------|
| `p-2` | 8px | 사이드바 영역, 버튼 소형 |
| `p-4` | 16px | 카드 내부, 리스트 아이템 |
| `p-6` | 24px | 사이드바 로고 영역 |
| `p-8` | 32px | 메인 콘텐츠 패딩 |
| `px-3 py-2` | 12px/8px | 네비게이션 링크 |
| `space-y-1` | 4px | 사이드바 메뉴 간격 |
| `space-y-4` | 16px | 폼 필드 간격 |
| `space-y-6` | 24px | 페이지 섹션 간격 |
| `gap-4` | 16px | 그리드 카드 간격 |
| `gap-6` | 24px | 레이아웃 그리드 간격 |

## 타이포그래피 (Typography)

Tailwind 기본 폰트 스케일 사용. 시스템 폰트 스택.

| 클래스 | 크기 | 사용처 |
|--------|------|--------|
| `text-xs` | 12px | Badge, 툴팁, 보조 정보 |
| `text-sm` | 14px | 기본 UI 텍스트, 테이블 셀, 폼 레이블 |
| `text-base` | 16px | 본문 기본값 |
| `text-lg` | 18px | 소제목 |
| `text-xl` | 20px | 사이드바 브랜드명 |
| `text-2xl` | 24px | 로그인 페이지 타이틀 |
| `text-3xl font-bold` | 30px | StatsCard 수치 |

### 폰트 굵기

| 클래스 | 사용처 |
|--------|--------|
| `font-medium` | 네비게이션 링크, 폼 레이블 |
| `font-semibold` | 프로필 이름, 소제목 |
| `font-bold` | 브랜드명, 페이지 타이틀, 통계 수치 |

## 애니메이션 (Animation)

| 이름 | 키프레임 | 지속시간 | 사용처 |
|------|----------|----------|--------|
| `accordion-down` | height: 0 → var(--radix-accordion-content-height) | 0.2s ease-out | Accordion 열기 |
| `accordion-up` | height: var(--radix-accordion-content-height) → 0 | 0.2s ease-out | Accordion 닫기 |
| `transition-all duration-300` | - | 300ms | 사이드바 너비 전환 |
| `transition-opacity duration-150` | opacity 0 → 100 | 150ms | 사이드바 툴팁 |
| `animate-spin` | 360도 회전 | 1s linear infinite | LoadingSpinner |
| `animate-pulse` | opacity 1 ↔ 0.5 | - | Skeleton |

## 다크 모드 전환

- `next-themes` 라이브러리 사용
- `ThemeProvider`에서 `attribute="class"` 설정 → `<html>` 태그에 `.dark` 클래스 토글
- Header의 `ThemeToggle` 컴포넌트: `Sun` / `Moon` 아이콘 전환
- 시스템 테마 자동 감지 (`defaultTheme="system"`)
