## SCR-001: 로그인

| 항목 | 내용 |
|------|------|
| **화면명** | 로그인 |
| **라우트** | `/login` |
| **접근 권한** | 공개 (인증 불필요) |
| **구현 상태** | 완료 |

**주요 기능:**
- 이메일/비밀번호 입력 폼
- 로그인 버튼 클릭 시 인증 처리
- 에러 메시지 표시 (잘못된 자격증명, 권한 없음)
- 로그인 성공 시 `/dashboard`로 이동

**사용 API:**
- `POST /admin/auth/login`
- `POST /admin/auth/verify` (페이지 로드 시 토큰 검증)

```
BlockPick Admin
├── Dashboard          /dashboard

│       └── Leaderboard /games/[id]/leaderboard  (미구현)

├── Storage            /storage       (미구현)

```
