# 1. 프로젝트 개요

## 1.1 프로젝트명

**BlockPick Admin Dashboard**

## 1.2 목적

BlockPick 플랫폼의 관리자용 웹 대시보드로, ADMIN 및 PARTNER 역할을 가진 운영자가 유저·게임·상품·블록체인·모니터링·파일 등 플랫폼 전반을 관리할 수 있는 통합 관리 도구를 제공한다.

## 1.3 범위

| 관리 영역 | 주요 기능 | 구현 상태 |
|-----------|-----------|-----------|
| 관리자 인증 | 로그인, 로그아웃, 토큰 관리 | 완료 |
| 유저 관리 | CRUD, 역할 변경, 검색/필터 | 완료 |
| 게임 관리 | CRUD, 상태 관리, 강제 종료, 정산 | 완료 |
| 상품 관리 | CRUD, 검색, 브랜드/카테고리별 조회 | 완료 |
| 게임-상품 통합 | 슬롯 관리, 지역 설정, 순서 재정렬 | 완료 |
| 블록체인 관리 | 컨트랙트 배포, 상태 관리, 검증 | UI 미완료 |
| 시스템 모니터링 | 헬스체크, 이벤트/트랜잭션 상태, 재발행 | UI 미완료 |
| 파일 관리 | 이미지 단일/다중 업로드 | UI 미완료 |
| 대시보드 | 종합 통계, 차트, 활동 내역 | 완료 |

## 1.4 이해관계자

| 역할 | 설명 | 시스템 접근 |
|------|------|-------------|
| **ADMIN** | 플랫폼 전체 운영자 | 모든 기능 접근 가능 |
| **PARTNER** | 브랜드/벤더 파트너 | 제한된 기능 접근 |
| **USER** | 일반 앱 사용자 | Admin Dashboard 접근 불가 |

## 1.5 기술 스택 개요

| 구분 | 기술 | 버전 |
|------|------|------|
| **Framework** | Next.js (App Router) | 16.x |
| **Language** | TypeScript | 5.x |
| **Styling** | Tailwind CSS | 3.x |
| **UI Components** | shadcn/ui | 최신 |
| **상태 관리** | TanStack React Query | 5.x |
| **테이블** | TanStack React Table | 8.x |
| **테마** | next-themes | 최신 |
| **아이콘** | Lucide React | 최신 |
| **API 통신** | REST (fetch 기반) | - |
| **인증** | JWT Bearer Token | HS256 |
