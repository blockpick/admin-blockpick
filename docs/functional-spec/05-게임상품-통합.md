# 게임-상품 통합 관리

> 버전: 1.0 | 작성일: 2026-02-24

---

## 1. 개요

- **목적**: 게임에 상품(경품) 슬롯을 연결·관리하고, 지역별 상품 설정 및 순서를 제어
- **대상 사용자**: ADMIN 역할 관리자
- **관련 화면**: `/games/[id]/products`

---

## 2. 화면 구성

| 화면 | 경로 | 주요 기능 |
|------|------|----------|
| 게임 상품 관리 | `/games/[id]/products` | 게임에 연결된 상품 목록, 추가/제거, 그랜드프라이즈 설정 |
| 상품 연결 다이얼로그 | `/games/[id]/products` (다이얼로그) | 기존 상품 선택 또는 신규 상품 생성 후 연결 |

---

## 4. 상세 기능

### 4.1 게임별 상품 목록 (`gameProductService.getGameProducts()`)

**설명**: 특정 게임에 연결된 모든 상품 슬롯 조회.

**Query Parameters:**
| 파라미터 | 기본값 | 설명 |
|---------|--------|------|
| `activeOnly` | false | true 시 active=true 슬롯만 반환 |

**GameProductDto 구조:**
```json
{
  "id": "uuid",
  "gameId": "uuid",
  "productId": "uuid",
  "product": { ...ProductDto },
  "isGrandPrize": false,
  "quantity": 1,
  "sequence": 1,
  "active": true,
  "createdAt": "ISO8601",
  "updatedAt": "ISO8601",
  "regions": [
    {
      "id": "uuid",
      "gameProductId": "uuid",
      "countryCode": "KR",
      "regionCode": "SEOUL",
      "productId": "uuid",
      "localizedName": "한국어 상품명",
      "localizedDesc": "한국어 설명",
      "imageUrl": "https://...",
      "currencyOverride": "CASH",
      "priceCents": 100000,
      "stock": 10,
      "shippingNotes": "배송 안내",
      "active": true
    }
  ]
}
```

---

### 4.2 통합 상품 추가 (`gameProductService.addProductToGame()`)

**설명**: 상품 생성 + 게임 슬롯 생성 + 지역 설정을 한 번의 API 호출로 처리.

**API:** `POST /admin/game-product-management/games/{gameId}/products/complete`

**Request Body (AddProductToGameRequest):**
```json
{
  "product": {
    "name": "상품명",
    "sku": "PROD-001",
    "brand": "브랜드",
    "description": "설명",
    "defaultImage": "https://...",
    "thumbnail": "https://..."
  },
  "gameProduct": {
    "gameId": "uuid",
    "isGrandPrize": false,
    "quantity": 1,
    "sequence": 1,
    "active": true
  },
  "regions": [
    {
      "countryCode": "KR",
      "regionCode": "SEOUL",
      "localizedName": "한국어 상품명",
      "priceCents": 100000,
      "stock": 10,
      "active": true
    }
  ]
}
```

**비즈니스 규칙:**
- regions 배열은 빈 배열 허용 (지역 설정 없이 연결)
- 상품 생성 실패 시 전체 트랜잭션 롤백

---

### 4.3 기존 상품 게임 연결 (`gameProductService.linkExistingProductToGame()`)

**설명**: 이미 존재하는 상품을 새 게임에 연결 (상품을 새로 만들지 않음).

**API:** `POST /admin/game-product-management/games/{gameId}/products/{productId}/link`

**Request Body (LinkProductToGameRequest):**
```json
{
  "gameProduct": {
    "gameId": "uuid",
    "isGrandPrize": false,
    "sequence": 1,
    "active": true
  },
  "regions": []
}
```

**UI 플로우 (LinkProductToGameDialog):**
1. "Add Product" 클릭
2. 기존 상품 검색/선택
3. 슬롯 설정 (수량, 순서, 그랜드프라이즈 여부)
4. 연결 완료

---

### 4.4 그랜드프라이즈 설정 (`gameProductService.setGrandPrizeProducts()`)

**설명**: 게임의 대상(그랜드프라이즈) 상품을 일괄 지정.

**API:** `PUT /admin/game-product-management/games/{gameId}/grand-prizes`

**Request Body:**
```json
{
  "gameProductIds": ["uuid-1", "uuid-2"]
}
```

**비즈니스 규칙:**
- 기존 그랜드프라이즈 설정을 새 목록으로 교체
- 빈 배열 전달 시 그랜드프라이즈 해제

---

### 4.5 상품 순서 재정렬 (`gameProductService.reorderGameProducts()`)

**설명**: 게임 내 상품 표시 순서(sequence)를 일괄 변경.

**API:** `PUT /admin/game-product-management/games/{gameId}/products/reorder`

**Request Body:**
```json
{
  "sequences": [
    { "gameProductId": "uuid-1", "sequence": 1 },
    { "gameProductId": "uuid-2", "sequence": 2 },
    { "gameProductId": "uuid-3", "sequence": 3 }
  ]
}
```

**비즈니스 규칙:**
- sequence는 1부터 시작하는 정수
- 모든 슬롯의 sequence를 한 번에 전달

---

### 4.6 지역 설정 관리

**설명**: 게임 상품 슬롯별로 국가/지역에 따른 별도 상품 정보를 설정.

**CreateGameProductRegionInput:**
```json
{
  "gameProductId": "uuid",
  "countryCode": "KR",
  "regionCode": "SEOUL",
  "localizedName": "한국어명",
  "localizedDesc": "한국어 설명",
  "imageUrl": "https://...",
  "currencyOverride": "CASH",
  "priceCents": 100000,
  "stock": 10,
  "shippingNotes": "배송 안내",
  "active": true
}
```

| 필드 | 설명 |
|------|------|
| `countryCode` | ISO 3166-1 alpha-2 국가 코드 (예: KR, US, JP) |
| `regionCode` | 지역 코드 (예: SEOUL, BUSAN) |
| `priceCents` | 가격 (센트 단위, 100000 = 1,000원) |
| `stock` | 재고 수량 |
| `currencyOverride` | 해당 지역 통화 설정 |

---

### 4.7 상품 완전 제거 (`gameProductService.removeProductFromGame()`)

**설명**: 슬롯과 연관된 모든 지역 설정까지 함께 삭제.

**API:** `DELETE /admin/game-product-management/game-products/{gameProductId}/complete`

**vs 슬롯 삭제 차이:**
| 방법 | 슬롯 삭제 | 지역 설정 삭제 |
|------|----------|--------------|
| 슬롯만 삭제 (DELETE .../game-products/{id}) | O | X (유지) |
| 완전 제거 (DELETE .../complete) | O | O |

---

### 4.8 게임 상품 통계 (`gameProductService.getGameProductStats()`)

**통계 카드 (UI):**
| 카드 | 설명 |
|------|------|
| 총 상품 수 | 연결된 슬롯 수 |
| 그랜드프라이즈 수 | isGrandPrize=true 슬롯 수 |
| 활성 상품 수 | active=true 슬롯 수 |
| 지역별 분포 | 지역 설정이 있는 슬롯 수 |

## 5. 에러 처리

| 에러 코드 | HTTP 상태 | 원인 | UI 처리 |
|----------|----------|------|--------|
| `GAME_PRODUCT_NOT_FOUND` | 404 | 존재하지 않는 슬롯 ID | "게임 상품을 찾을 수 없습니다." |
| `PRODUCT_ALREADY_LINKED` | 409 | 이미 연결된 상품 | "이미 이 게임에 연결된 상품입니다." |
| `INVALID_SEQUENCE` | 400 | 잘못된 순서 값 | "순서 값이 올바르지 않습니다." |
| `REGION_NOT_FOUND` | 404 | 존재하지 않는 지역 설정 | "지역 설정을 찾을 수 없습니다." |
