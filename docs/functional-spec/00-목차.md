## 미구현 백엔드 API (향후 필요)

아래 기능들은 백엔드 API 자체가 없으며 별도 개발이 필요합니다.

| 기능 | 설명 |
|------|------|
| 유저 페이지네이션 | GET /admin/users 현재 전체 반환 |
| 유저 밴(차단) 관리 | isBan 토글 API 없음 |
| 유저 재화 수동 조정 | 관리자 직접 충전/차감 API 없음 |
| 공지사항 관리 | CRUD 없음 |
| 배너/이벤트 관리 | CRUD 없음 |
| 당첨자 관리 | 목록/발송상태 관리 없음 |
| 결제/환불 관리 | 결제 내역 조회 없음 |
| 운영 로그 | 관리자 활동 Audit Trail 없음 |

## 기술 스택 참고

- **Frontend**: Next.js (App Router), TypeScript
- **Auth**: JWT Bearer Token (Access 24h / Refresh 7일)
- **API Base**: `https://api-dev.blockpick.net`
- **인증 헤더**: `Authorization: Bearer <accessToken>`
- **페이지네이션**: Spring Page 형식 (page 0-based, size 기본 20)
