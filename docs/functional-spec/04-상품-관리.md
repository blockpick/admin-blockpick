# 상품 관리

> 버전: 1.0 | 작성일: 2026-02-24

---

## 1. 개요

- **목적**: BlockPick 게임에 사용되는 상품(경품)의 생성, 조회, 수정, 삭제 및 카테고리 관리
- **대상 사용자**: ADMIN 역할 관리자
- **관련 화면**: `/products`

---

## 2. 화면 구성

| 화면 | 경로 | 주요 기능 |
|------|------|----------|
| 상품 목록 | `/products` | 검색/필터/페이지네이션, 통계 카드, CRUD 액션 |
| 상품 상세 다이얼로그 | `/products` (다이얼로그) | 상품 전체 정보 표시 |
| 상품 생성 다이얼로그 | `/products` (다이얼로그) | 신규 상품 생성 (이미지 업로드 포함) |
| 상품 수정 다이얼로그 | `/products` (다이얼로그) | 기존 상품 정보 수정 |

---

## 4. 상세 기능

### 4.1 상품 목록 조회 (`productService.getProducts()`)

**설명**: 페이지네이션 지원 상품 목록. Spring Page 형식 응답.

**Query Parameters:**
| 파라미터 | 기본값 | 설명 |
|---------|--------|------|
| `page` | 0 | 페이지 번호 (0-based) |
| `size` | 20 | 페이지 크기 |
| `sortBy` | `createdAt` | 정렬 기준 |
| `sortDirection` | `DESC` | 정렬 방향 |

**ProductDto 구조:**
```json
{
  "id": "uuid",
  "sku": "PROD-001",
  "brand": "브랜드명",
  "name": "상품명",
  "description": "상품 설명",
  "defaultImage": "https://blockpick-dev-storage.s3.../products/image.jpg",
  "thumbnail": "https://blockpick-dev-storage.s3.../products/image_thumb.jpg",
  "metadata": {},
  "createdAt": "2026-01-27T00:00:00+09:00",
  "updatedAt": "2026-01-27T00:00:00+09:00"
}
```

**UI 필터 옵션:**
| 필터 | 옵션 |
|------|------|
| 브랜드 | 전체 / 브랜드 목록 |
| 카테고리 | 전체 / 카테고리 목록 |
| 상태 | 전체 / 활성 / 비활성 |
| 검색 | 상품명/SKU/브랜드 키워드 |

**테이블 컬럼:** Image, Name, Brand, Category, SKU, Status, Created, Actions

---

### 4.2 상품 생성 (`productService.createProduct()`)

**설명**: 신규 상품을 생성. 이미지 업로드 시 Storage API 선행 호출 필요.

**입력 (CreateProductInput):**
```json
{
  "name": "상품명",
  "sku": "PROD-001",
  "brand": "브랜드명",
  "description": "상품 설명",
  "thumbnail": "https://...",
  "defaultImage": "https://...",
  "metadata": {}
}
```

| 필드 | 타입 | 필수 | 규칙 |
|------|------|------|------|
| `name` | String | O | 상품명 |
| `sku` | String | X | SKU 코드 (고유값) |
| `brand` | String | X | 브랜드명 |
| `description` | String | X | 상품 설명 |
| `thumbnail` | String | X | 썸네일 이미지 URL (Storage API 업로드 후) |
| `defaultImage` | String | X | 기본 이미지 URL (Storage API 업로드 후) |
| `metadata` | JSON | X | 추가 메타데이터 (자유 형식 JSON) |

**이미지 업로드 플로우:**
1. 파일 선택
2. `POST /admin/storage/upload/image` 호출 → URL 반환
3. 반환된 URL을 `thumbnail` 또는 `defaultImage` 필드에 설정
4. `POST /admin/products` 호출

**비즈니스 규칙:**
- SKU는 고유값 (중복 시 에러)
- 이미지는 JPG, JPEG, PNG, GIF, WEBP 지원
- 생성 성공 시 목록 새로고침 및 성공 토스트

---

### 4.3 상품 상세 조회 (`productService.getProductById()`)

**설명**: 상품 상세 다이얼로그에 표시할 전체 정보 조회.

**표시 정보:**
- 기본 정보: 이미지, 이름, 설명, 브랜드, SKU
- 메타데이터: JSON 형식 추가 정보
- 생성/수정일

**액션 버튼:** 수정, 삭제

---

### 4.4 상품 수정 (`productService.updateProduct()`)

**설명**: 기존 상품 정보 부분 업데이트. `id` 필드 필수.

**입력 (UpdateProductInput):**
```json
{
  "id": "uuid (필수)",
  "sku": "PROD-001",
  "brand": "새 브랜드",
  "name": "새 상품명",
  "description": "새 설명",
  "defaultImage": "https://...",
  "thumbnail": "https://...",
  "metadata": {}
}
```

**비즈니스 규칙:**
- `id` 외 모든 필드 선택적
- 이미지 교체 시 Storage API 업로드 후 URL 업데이트

---

### 4.5 상품 삭제 (`productService.deleteProduct()`)

**설명**: UUID로 상품 삭제.

**플로우:**
1. Actions에서 "Delete" 클릭
2. 확인 다이얼로그 표시
3. 확인 시 DELETE 요청
4. 성공 시 목록에서 제거

**비즈니스 규칙:**
- 게임에 연결된 상품 삭제 시 서버에서 에러 반환 가능
- 삭제 전 연결된 게임 여부 안내 권장

---

### 4.6 카테고리 목록 조회 (`productService.getCategories()`)

**설명**: 필터링 UI에서 사용할 카테고리 목록 조회.

**비즈니스 규칙:**
- 페이지 로드 시 한 번 조회 후 캐싱
- 카테고리 필터 드롭다운에 동적으로 표시

---

### 4.7 브랜드별 상품 조회 (`productService.getProductsByBrand()`)

**설명**: 특정 브랜드의 상품만 필터링 조회.

**API:** `GET /admin/products/brand/{brand}`

---

### 4.8 SKU로 상품 조회 (`productService.getProductBySku()`)

**설명**: SKU 코드로 상품 단건 조회. 게임-상품 연결 시 상품 확인에 활용.

**API:** `GET /admin/products/sku/{sku}`

---

### 4.9 상품 통계 (`productService.getProductStats()`)

**통계 카드 (UI):**
| 카드 | 설명 |
|------|------|
| 총 상품 수 | 전체 상품 수 |
| 활성 상품 | 활성 상태 상품 수 |
| 브랜드 수 | 등록된 고유 브랜드 수 |
| 카테고리 수 | 등록된 고유 카테고리 수 |

## 5. 에러 처리

| 에러 코드 | HTTP 상태 | 원인 | UI 처리 |
|----------|----------|------|--------|
| `PRODUCT_NOT_FOUND` | 404 | 존재하지 않는 상품 ID | "상품을 찾을 수 없습니다." |
| `SKU_ALREADY_EXISTS` | 409 | SKU 중복 | 생성 다이얼로그 인라인 에러 |
| `PRODUCT_IN_USE` | 400 | 게임에 연결된 상품 삭제 시도 | "게임에 연결된 상품은 삭제할 수 없습니다." |
| `INVALID_FILE_TYPE` | 400 | 지원하지 않는 이미지 형식 | "JPG, PNG, GIF, WEBP 파일만 업로드 가능합니다." |
